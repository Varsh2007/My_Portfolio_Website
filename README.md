# My Portfolio Website

A personal portfolio website showcasing my projects, skills, and professional journey.

## 🌟 About

This is my first web app that I have successfully created and deployed. It serves as a digital resume and project showcase, demonstrating my web development skills and experience.

## 🚀 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Project Showcase** - Display of my best projects with descriptions and links
- **Skills Section** - Highlighting my technical expertise and tools
- **Contact Form** - Easy way for visitors to get in touch
- **Clean UI** - Modern and professional user interface

## 🛠️ Tech Stack

- **Frontend**: HTML & CSS
- **Hosting**: GitHub Pages

## 📁 Project Structure

```
My_Portfolio_Website/
├── index.html          # Main landing page
├── about_me.html       # Personal background
├── education.html      # Academic background
├── skills.html         # Technical expertise
├── projects.html       # Project showcase
├── contact_form.html   # Contact form page
├── css/                # Stylesheets
│   └── style.css
├── img/                # Images and icons
└── README.md           # This file
```

## 🔧 Customization

To personalize this portfolio:

1. Update HTML files with your own content
2. Modify colors and styling in `css/style.css`
3. Add your projects to `projects.html`
4. Update contact information in `contact_form.html`
5. Add your photos and media to `img/` folder

## 📚 Sections

- **Home** - Introduction and call-to-action
- **About** - Personal background and professional summary
- **Skills** - Technical skills and expertise
- **Education** - Academic background
- **Projects** - Showcase of completed projects
- **Resume** - Link to digital resume
- **Contact** - Contact form and social links
